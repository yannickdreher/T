# T - SSH Terminal Client

A modern, cross-platform SSH terminal client built with Avalonia UI.

## Features

- 🖥️ Terminal emulator (VT100/xterm-256color)
- 📁 SFTP file browser with Auto-upload on file save
- 🔐 Multiple authentication methods (password, private key)
- 📂 SSH Session management with folder organization

## Installation

### Windows (ClickOnce)

1. **Install .NET 10 Runtime** (required):
   - Download from [Microsoft .NET Download Page](https://dotnet.microsoft.com/download/dotnet/10.0)
   - Install the **.NET Desktop Runtime 10.0** (x64)

2. **Install T**:
   - Visit the [latest release](https://yannickdreher.github.io/T/latest/)
   - Click the **Install** button
   - Follow the ClickOnce installer prompts

**Auto-Updates**: The application automatically checks for updates every 7 days in the background.

### Manual Installation

Download the latest `windows-x64-installer.zip` from the [Releases](https://github.com/yannickdreher/T/releases) page and extract it.
